# Diamond Price Predictor

A machine learning application that predicts diamond prices based on various features like carat, cut, color, clarity, and dimensions.

## Features
- Predict diamond prices using ML models
- Clean and modern web interface
- REST API support

## Usage
Enter the diamond characteristics in the form and click "Predict Price" to get an estimated value.
